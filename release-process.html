The release process

Before releasing

There are some facts to check before you start to release your project:

  • make sure your groupId is `com.zenika` or is not set (will use the current groupId)
  • make sure you set your License in your pom and in the LICENSE file
    • The license of _this_ project is Apache 2.0, so do not forget to mention it
  • make sure you have a correct gpg key pair for your @zenika.com email
    • you may have to upload your gpg key to a gpg key hosting site like http://pgp.mit.edu/
