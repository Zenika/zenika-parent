Welcome to Zenika Parent!

The projects using this pom as their parent will have:

  • easy configuration to deploy their artefacts to maven-central,
  • easy documentation publication to github pages,
  • easy artifacts upload to github downloads section

Usage

To be able to use this POM, make sure to add the following code to the POM of your project

<project>
  [...]
  <parent>
    <groupId>com.zenika</groupId>
    <artifactId>zenika-parent</artifactId>
    <version>1-SNAPSHOT</version>
  </parent>
  [...]
</project>
