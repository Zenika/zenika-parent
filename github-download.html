Upload artefacts to github

  • to be able to upload your project artifacts to downloads section, make sure you set your github token correction in a `github.global.oauth2Token` property.
    • cf https://github.com/github/maven-plugins#core-configuration

    You can upload your artifacts to Github with the following command:

    $ mvn clean package ghDownloads:upload

    You will see your artefacts uploaded to https://github.com/Zenika/zenika-parent/downloads.

    Your artifacts will be uploaded on deploy phase too if you set the property `inherit.parent.gh-download` to `true`
